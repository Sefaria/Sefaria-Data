עזרא פרק-א

{א} העיר. מל' התעוררות : רוח. רצון כמו הנני נותן בו רוח (מ''ב יט) : ויעבר קול. ענין הכרזה כמו ויעבירו קול במחנה (שמות לו) : {ב} פקד. צוה כמו פקודי ה' (תהלים י''ט) : {ד} וברכוש. כן יקרא כל שוה כסף כמו ואת כל רכושו (בראשית יד) : {ה} ראשי האבות. ראשי המשפחות : לכל העיר. את כל העיר הלמ''ד במקום את וכן הרגו לאבנר (ש''ב ו) : רוחו. רצונו : {ו} ובמגדנות. המלה ההיא יורה על דבר המעולה בין בפירות בין במלבושים בין בכלים וכן ומגדנות לחזקיה (דה''ב ל''ב) : לבד על. הוא כפול לתוספ' ביאור וכן הרק אך במשה (במדב' יב) : {ח} על יד. ביד על במקום בי''ת וכן על צבאותם (שמות יב) ורבים כמוהו : הגזבר. הפקיד : {ט} אגרטלי. שם כלי ינתן בו המים לנטילת הידים : מחלפים. סכינים והוא מל' כליל יחלוף (ישעיה ב) ע''ש שבהם שוחטים ומחליפים מחיים למיתה ועש''ז קראו חז''ל לצדדי האולם מזה ומזה בית החליפות (מדות פ''ד) כי שם גנזו הכהנים סכיניהם : {י} כפורי. מזרקות וע''ש שהכהן הוורק היה מקנח אצבעו מן הדם בשפת המזרק לזה קרוי כפור שהוא ענין קנוח ובדרז''ל ובעי לכפורי ידיה (גיטין נו) : משנים. מל' שניים ור''ל כפולים הראוי להשתמש בהן בשתי עבריהן :

עזרא פרק-ב

{א} . : {סב} המתיחשים. ענין יחוס הוא הודעת סדר הדורות מן המשפחות ממי יצאו ומי אביהם : ויגואלו. ענין טנוף ונמאס כמו לחם מגואל (מלאכי א) : {סד} רבוא. הוא עשרת אלפים : {סו} פרדיהם. הם הבאים מן הסוס והחמור : {סח} מכונו. מל' כן ובסיס : {סט} דרכמונים. שם מטבע ידוע אצלם : מנים. שם משקל ליטרא :

עזרא פרק-ג

{א} ויגע. כמו והגיע : {ג} מכונותיו. מל' כן ובסיס : {ז} לצדונים ולצורים. אנשי צידון וצור : כרשיון. הוא מל' רשות הידוע בדרז''ל : {ח} לנצח. ענינו נצוח השיר ע''ש שדרך בעלי השיר לנצח זא''ז בהרמת הקול ובהכרעת הנעימה : {ט} ואחיו. חבריו ורעיו : לנצח. ענינו נגישה וכפיה והיא מל' נצחון והתגברות : {י} במצלתים. עשוי הוא משני כלי נחושת להשמיע קול גדול בהקשתן זה בזה וכן בצלצלי שמע (תהלים קו) : {יא} ויענו. ענין הרמת קול כמו וענו הלוים וגו' קול רם (דברים כז) :

עזרא פרק-ד

{א} צרי. אויבי : {ד} מרפים. מל' רפיון : לבנות. כמו מלבנות : {ה} וסוכרים. כמו ושוכרים בשין : {ו} שטנה. ענין קטרוג : {ז} כנותיו. ענינו רעיו וחבריו ובדרז''ל שבת ועבוד' כוכבים בכנותהון (כריתות ג) כן גרס הערוך ור''ל בכל דיניהם המתחבר אליהם : הנשתון. ענינו מפורש ואין לו דומה כי אם בזה הספר : ומתרגם. גם הוא ענין מפורש ובדרז''ל מאן דמתרגם לי חבית (ב''מ מא) : {ח} בעל טעם. ענינו היודע מיטב הדבור במליצה נאותה וכן וטע' זקנים יקח (איוב י''ב) : אגרא. כן יקרא כתב השלוח ע''ש שמאוסף בו דברים הרבה והוא מל' אוגר בקיץ (משלי י) : כנמא. ענין אמירה וכן וינאמו נאם (ירמיה כ''ג) : {יא} פרשגן. ענין טופס הדבר וענינו כי משנה תורת משה (יהושע ח) ת''י פרשגן אורייתא : {יב} ושוריא. חומות וכן אדלג שור (תהלים י''ח) : שכלילו. ענין גמר והשלמה וכת''א על ויכלו השמים (בראשית ב') : ואושיא. ענין יסוד כמו לאשישי קיר חרשת (ישעיה ט''ז) : יחיטו. מל' חוט ר''ל יחברו כאלו תפרם בחוט ע''י המחט : {יג} מנדה. זה מס המלך המוטל על כל בני המדינה ביחד : בלו. הכסף גולגלתא. והלך. הוא תשודה המביאים למלך בעת יעבור אליהם : ואפתום. הוא ענין הכנסה ואין לו דומה : {יד} מלח. ענין חרבן כמו ארץ מלחה (ירמיה י''ז) : וערות. מל' ערוה ר''ל בזיון : אריך. מענין יפה וכשר ובדרז''ל מאה פלפלין במאה ועשרים ואריך (ב''מ עה) ור''ל והדבר יפה וישר ואין בו משום רבית : {טו} יבקר. יחפש כמו אבקר את צאני (יחזקאל ל''ד) : ואשתדור. ענינו סרבנות ומרידה והוא מלשון שדר שהוא שליחות בלשון ארמי כי כן דרך המורדים לשלוח שלוחים לשאול עזר להחזיק ידם : {יט} טעם. ענינו צווי וגזירה ועצה כמו מטעם המלך (יונה ג) : {כב} וזהירין. הוא מל' זהירות הידוע בדרז''ל שהוא כענין זריזות אלא שזריזות הוא להשמר שלא תבוא הדבר לידו וזהירות הוא להשמר ממנה בעת באה לידו : שלו. משגה ושגגה וכן על השל (ש''ב ו') :

עזרא פרק-ה

{ב} ושריו. ענין התחלה כי וזה החילם (בראשית י''א) ת''א ודין דשריאו : מסעדין. מל' סעד וסיוע : {ג} ושתר בוזני. כך שמו : ואשרנא. כמו ושוריא ור''ל החומות : {ח} אספרנא. הוא ענין מהירות רב ואין לו דומה כי אם בזה הספר :

עזרא פרק-ו

{ב} באחמתא. הוא מל' וחמת מים (בראשית כ''א) והוא נאד של עור : מגלה. ספר : {ג} מסובלין. ענין משא כבדה כמו והנושאים בסבל (נחמיה ד) : {ד} נדבכין. שורות ובדרז''ל קורין וכו' ובראש הנדבך (ברכות ט''ו) : {ז} ולשבי. הלמ''ד הוא במקום את ור''ל ועם סבי : {ט} חשחן. כמו חוששין וכן לא חשחין אנחנא (דניאל ג) : שלו. ענין משגה ושכחה כמו על השל (ש''ב ו') : {יא} יתנסח. ענין עקירה כמו בית גאים יסח ה' (משלי ט''ו) : וזקיף. הוא העץ שתולין עליו ובדרז''ל דאית ליה זקיפתא וכו' זקיף ביניתא (ב''מ נ''ט) : יתמחא. יושם ובדרז''ל המחוהו אצל חנוני (שם קי''א) : {יב} ימגר. ענין הפלה כמו וכסאו לארץ מגרת (תהלים פ''ט) : {טו} ושיציא. ענין גמר והשלמה כי כאשר כלו (בראשית מ''ג) ת''א כד שציאו : {טז} חנכת. התחלת דבר מה לעמוד בה משם והלאה נקרא חנוך בלשון המקרא : {יח} בפלוגתהון. ענין חלוקה : {כב} והסב. מל' סבוב ור''ל הטה ופנה :

עזרא פרק-ז

{ו} סופר. ר''ל חכם ובדרז''ל סופר מברך ובור יוצא (ברכות מ''ה) : מהיר. הוא הפוך מן קשה ההבנה : {ז} והשוערים. שומרי השערים. והנתינים. הגבעונים. (יב) גמיר. כולה כי כליל תקטר (ויקרא ו) ת''א גמיר מתסק : {יג} מתנדב. ענין רצון כי כל המתנדב מתנדב בדצון : {כ} השהות. כמו חששות והוא ענין צורך : יפל. ענין נטיה ודוגמתו כי יפול לא יוטל (תהלים ל''ז). (כב) : בתין. שם מדת הלח והיא ג' סאין : {כג} אדרזדא. הוא ענין זריזות ואין לו דומה :

עזרא פרק-ח

{כ} נקבו. נתפרשו וכן ונוקב שם (ויקרא כ''ד) : {כב} ופרשים. הם רוכבי הסוסים בהרגל רב : {כג} ויעתר. ענין קבלת תפלה וכן ויעתר לו (בראשית כ''ה) : {כז} כיפורי. מזרקות : לאדרכונים. שם מטבע ידוע אצלם : מוצהב. מראה כעין גוון הזהב וכן שער צהוב (ויקרא יג) : {כט} שקדו. ענין מהירות כמו כי שוקד אני (ירמיה א') : {לא} ואורב. שונא היושב במארב להסתר : {לה} צפירי. שעירי :

עזרא פרק-ט

{א} לאחשדרפני. מין שררה ידוע אצלם : ופחוות. מל' פחה ומושל : {ב} נבדלו. ענין הפרשה : {ג} והתערבו. מל' תערובות ובלבול : והסגנים. מין שררה : במעל. ענין חטא ופשע : {ד} ומעילי. הוא מין מלבוש עליון : ואמרטה. ענין תלישה כמו ואיש כי ימרט ראשו (ויקרא י''ג) : משומם. משותק ונאלם כמו משמים בתוכם (יחזקאל ג') : {ה} חרד. ענין מהירות כמו ויחרדו זקני העיר (ש''א י''ו) : {ז} ונכלמתי. מלשון כלימה ובושה : להרים. לרומם ולהגביה : ואשמתנו. מל' אשמה ופשע : {ט} תחנה. חנינה וחן וכן לבלתי היות להם תחנה (יהושע י''א) : פליטה. ענין הצלה ושיור : יתד. מסמר : מחיה. מל' חיות : {י} חרבותיו. מלשון חורבה : גדר. כותל מקיף : {יב} נדה. מל' נדידה ורחוק : {יד} חשכת. מנעת כמו ולא חשכת (בראשית כ''ב) : {טו} להפר. לבטל : להתחתן. מלשון חתון : תאנף. מל' אף וחמה :

עזרא פרק-י

{א} וכהתודותו. מל' וידוי : ומתנפל. נופל ומשתטח בארץ : {ב} מקוה. מל' תקוה ותוחלת : {ג} והחרדים. הממהרים : {ד} קום. הוא ענין זרוז וכן קום עבור (יהושע א) : {ו} לשכת. ענין חדר מה : {ח} יחרם. ענין חורבן ואבדון : {ט} מרעידים. מל' רעדה ורתת : {יא} תודה. הודאה : {יד} מזומנים. ענין הכנה וקביעות כמו בעתים מזומנות (נחמיה יג) :
