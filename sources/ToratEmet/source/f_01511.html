זכריה פרק-א

{ו} זמם. ענין מחשבה כמו זממו אל תפק (תהלים קמ) : {ח} ההדסים. שם אילן המריח : במצולה. מקום רבוי המים עניינו כמו ותשליכני מצולה (יונה ב) : שרוקים. מצויירים בצבעים מחולפים ודוגמתו בדרז''ל אשה לא תעביר שרק על פניה (שבת צה) : {יא} יושבת. רצה לומר יושבת בשלוה : ושוקטת. על השקט ומרגוע : {יב} ויען. ענין הרמת קול כמו וענית ואמרת (דברים כו) : זעמתה. מל' זעם וכעס : {יג} נחומים. מל' נחמה : {יד} קנאתי. ענין כעס כמו הם קנאוני בלא אל (שם לב) : {טו} השאננים. מל' שאנן והשקט : {טז} וקו. ענינו חוט המשקולת של הבנאי וכן קו לקו (ישעיה כח) : ינטה. מל' הטיה : {יז} תפוצנה. ענין פזור כמו ומאלה נפצה כל הארץ (בראשית ט) :

זכריה פרק-ב

{א} קרנות. מל' קרן : {ב} זרו. פזרו כמו ואתכם אזרה בגוים (ויקרא כו) : {ג} חרשים. אומנים כמו חרש עצים (ישעיה מד) : {ד} כפי. כשיעור כמו איש כפי אכלו (שמות יב) : לידות. ענין הפלה והשלכה כמו ידו אליה (ירמיה נ) : הנושאים. ענין הרמה : לזרותה. לפזרם : {ה} חבל מדה. קו המדה : {ו} אנה. להיכן כמו אנה אנחנו עולים (דברים א) : {ח} רוץ. ענין מהירת ההליכה : הלז. הזה וכן מה הציון הלז (מ''ב כג) : פרזות. ר''ל מבלי חומה סביב וכן ראש פרזיו (חבקוק ג) : {י} הוי. הוא ענין קריאה וכן הוי כל צמא (ישעיה נה) : ונוסו. ענין בריחה ומהירות ההליכה : רוחות. פאה ועבר : {יא} המלטי. ענין הצלה : בת בבל. עדת בבל : {יב} השוללים. ענין בזה : בבבת עינו. הוא גלגל העין וכן כאישון בת עין (תהלים יז) : {יג} מניף. ענין הרמה כמו והניף הכהן (ויקרא כג) : {טו} ונלוו. ענין התחברות כמו ונלוה הגר (ישעיה יד) : {יז} הס. ענין שתיקה כמו ויהס כלב (במדבר יג) : כל בשר. כל אדם : נעור. מלשון הערה והקיצה כמו אשר יעור משנתו (לקמן ד') : ממעון. ענין מדור כמו ה' מעון אתה היית לנו (תהלי' צ) :

זכריה פרק-ג

{א} השטן. מלאך מקטרג : לשטנו. ענין קטרוג וכן כתבו שטנה (עזרא ד') : {ב} יגער. ענין צעקה נזיפה : אוד. הוא העץ שמנדנדים בו האש כמו כאוד מוצל משרפה (עמוס ד') : {ג} צואים. ר''ל מלוכלכים ולפי שהצואה הוא לכלוך קורא לבגדים מלוכלכים בגדים צואים : {ד} ויען. ענין הרמת קול כמו וענית ואמרת (דברים כו) : מחלצות. כן יקראו מין מלבושים נאים וכן המחלצות והמעטפות (ישעיה ג) : {ה} צניף. מל' מצנפת העשוי לסבב את הראש : טהור. ר''ל ברור ונקי : {ו} ויעד. ענין התראה כי רוב המתרים מתרים בעדים למען לא יכחש לאחר זמן : {ז} תדין. מל' דין והוא ענין היישרת הדבר : מהלכים. מל' הלוך וטיול : {ח} מופת. ענין נס ואות : {ט} שבעה עינים. ר''ל הרבה וכן שבע יפול צדיק (משלי כד) : מפתח פתוחה. ענין חקיקת הציור כמו ויודע לפתח פתוחים (ד''ה ב' ב') : ומשתי. ענין הסרה כמו לא ימיש עמוד הענן (שמות יג) :

זכריה פרק-ד

{א} ויעירני. מל' הערה והקצה : {ב} וגולה. ספל עגול וכן וגולת הכותרות (מ''א ז') : נרותיה. כעין בזיכין על השמן והפתילה כמו ועשית את נרותיה (שמות כה) : מוצקות. מל' יציקה והם צינורות ודרך שם ניצוק ונמשך השמן : {ד} ואען. ענין הרמת קול : {ו} בחיל. ענין צבאות עם : ברוחי. ענין רצון : {ז} למישור. מל' ישר ושוה : הראשה. ר''ל טובה וחשובה וכן וראשית שמנים ימשחו (עמוס ז') : תשואות. ענין המית קול כמו תשואות נוגש (איוב לט) : {ט} תבצענה. ענין השלמה כמו כי יבצע ה' (ישעיה י) : {י} בז. מל' בזיון : האבן הבדיל. הוא האבן התלוי בקו המשקולת וע''י מיישר הבנאי את הבנין ועל שע''פ רוב דרכם לעשותה מאבן אמר האבן הבדיל וכן אבן העופרת (לקמן ה) : משוטטים. ענין התנועה הנה והנה וכן שוטטו בחוצות ירושלים (ירמיה ה) : {יב} שבלי. מל' השבלים הטובות (בראשית מא) ובדרך שאלה קורא כן ענפי הזית לפי שהיו מלאים גרגרי זיתים כמו השבולת המלאה גרגרי תבואה : ביד. ר''ל בתוך וכן ביד כל אדם יחתום (איוב ל''ז) : צנתרות. הוא שם כלי מה : המריקים. ענין הזלה ושפיכה כמו על הארץ יריקו (קהלת י''א) : הזהב. ר''ל שמן מזוקק ונקי כזהב וכן מצפון זהב יאתה (איוב ל''ז) : {יד} היצהר. הוא השמן כמו תירוש ויצהר (דברים כ''ח) : על אדון. אצל אדון :

זכריה פרק-ה

{א} מגלה. אגרת על שם שגוללין אותה : עפה. מל' עפיפה ופריחה : {ג} האלה. ענין קללה : נקה. מל' נקי : {ד} ולנה. ענין התמדה כמו צדק ילין בה (ישעיה א) : וכלתו. מל' כליון והשמדה : {ה} שא. הרם : נא. עתה : {ו} האיפה. שם מדה : {ז} ככר. שם משקל : נשאת. ענין הרמה : {ט} החסידה. שם עוף : ותשנה. כמו ותשאנה באלף : {י} אנה. להיכן : {יא} שנער. היא בבל : והוכן. מכונתה. מל' הכנה וקביעות :

זכריה פרק-ו

{א} מרכבות. העגלות שהסוסים מושכים אותם : {ג} ברודים. בעלי כתמים לבנים כעין הברד וכן נקודים וברודים (בראשית ל''א) : אמוצים. בעלי מראה דשן כן ת''י קטמנין והוא דשן בל' ארמי : {ה} ארבע. כמו לארבע ותחסר הלמ''ד וכן היו בן שאול (ש''ב ד') והראוי לבן שאול : רוחות. פאה ועבר : מהתייצב. ענין עמידה : על אדון. לפני אדון וסמוך לו : {ח} אותי. ר''ל אלי : הניחו. מל' הנחה והשקטה : רוחי. ענין כעס כמו ינחל רוח (משלי י''א) : {יא} עטרות. כתרות : {יב} איש. ר''ל גדול וחשוב : ומתחתיו. ממקומו ר''ל מזרעו : {יג} הוד. ענין הדר ויופי : כהן על כסאו. ר''ל לפני כסאו וכן לרצים הנצבים עליו (ש''א כב) :

זכריה פרק-ז

{ב} שראצר. שם איש וכן רגם מלך : לחלות. ענין תפלה כמו ויחל משה (שמות ל''ב) : {ג} לבית ה'. בבית ה' באה הלמ''ד במקום בי''ת וכן ישבת לכסא (תהלים ט') : הנזר. ענין הפרשה כמו ימי נזרו (במדבר ז') : כמה. ענינו הרבה : {ז} והנגב. ערי הנגב : ושפלה. העמק : {יא} וימאנו. לא רצו : להקשיב. ענין שמיעה : סוררת. מל' הסרה : {יב} שמיר. הוא הברזל החזק וכן כשמיר חזק מצור (יחזקאל ג) : ברוחו. במלאכו וכן ה' אלהים שלחני ורוחו (ישעיה מ''ח) : {יד} ואסערם. מל' סערה והוא הרוח החזק : נשמה. מל' שממון : וישימו. ועשו :

זכריה פרק-ח

{ב} קנאתי. ענין כעס : {ד} משענתו. מטהו שנשען עליה : {ה} משחקים. מל' שחוק ושמחה : {ז} מבוא השמש. שקיעת השמש כמו ממזרח שמש עד מבואו (תהלים קי''ג) : {י} שכר. ענין ריוח : ושכר הבהמה. שכר עבודת הבהמה : הצר. השונא : ואשלח. ענין גרוי והשסה וכן ושן בהמות אשלח בם (דברים ל''ב) : {יב} יבולה. תבואתה כמו ארץ נתנה יבולה (תהלים ס''ז) : טלם. מל' טל : {יד} זממתי. ענין מחשבה כמו כאשר זמם (דברים יט) : נחמתי. ענין הפוך מחשבה : {טז} בשעריכם. שם מקום המשפט כמ''ש אל זקני העיר השערה (שם כב) : {כא} לחלות. ענין תפלה כמו ויחל משה (שמות לב) : ולבקש. מל' בקשה ושאלה : {כב} עצומים. ענין רבוי כמו עצמו לי אלמנותיו (ירמיה טו) : {כג} יחזיקו. ענין אחיזה כמו ויחזק בכנף מעילו (ש''א טז) : לשונות הגוים. ר''ל עכו''ם בעלי לשונות נפרדות : בכנף. הוא שולי הבגד : יהודי. כל ישראל נקראים בשם יהודים לפי שהמלוכה היה משבט יהודה וכן תהיה לעתיד :

זכריה פרק-ט

{א} משא. ענין נבואה ולתוספת ביאור אמר דבר ה' : חדרך. הוא מקום אצל דמשק כן ארז''ל : {ב} תגבל. מל' גבול : {ג} ותבן. מל' בנין : מצור. ענין מבצר וחוזק : ותצבר. ענין כרי וגל כמו שני צבורים (מ''ב י) : וחרוץ. מין זהב טוב כמו בירקרק חרוץ (תהלים סח) : {ד} יורישנה. ענין שלוח וגרושין, כמו ויורש ה' מפניהם (יהושע כג) : והכה. ר''ל יטביע : חילה. מל' חיל וצבאות עם : {ה} ותחיל. מל' חיל וחלחלה : הוביש. מל' בושה : מבטה. מל' הבטה והסתכלות : {ו} גאון. ענין רוממות וממשלה : {ז} ושקוציו. מל' שקץ ושרץ : כאלוף. ענין למוד כמו פן תאלף ארחותיו (משלי כב) : כיבוסי. היא ירושלים : {ח} מצבה. כמו מצבא באל''ף : נוגש. דוחק ולוחץ : {ט} הריעי. מל' תרועה : עני. ר''ל מוכנע ועניו וכן עם עני תושיע (תהלים יח) : עיר. כן יקרא החמור בעודו קטן כמו ועירים עשרה (בראשית לב) : אתונות. כן תקרא נקבת החמור וכן ואחת האתונות (מ''ב ד) : {י} אפסי. קצות כי בקצה הארץ היא אפס וכלה : {יא} שלחתי. כמו אשלח עבר במקום עתיד כדרך הנבואה בהרבה מן המקומות : אסיריך. ענין קשירה : {יב} לבצרון. מל' מבצר : משנה. ענין כפל כמו וכסף משנה (בראשית מג) : אשיב. מל' השבה ומענה : {יג} דרכתי. הדרך לדרוך ברגל על הקשת למותחו היטב לירות למרחוק : קשת. בקשת : מלאתי. זה הלשון יאמר על אחיזת הקשת בכח כמו ויהוא מלא ידו בקשת (מ''ב ט) : ועוררתי. מל' התעוררות : {טו} וכבשו. ענין האחיזה בחזקה כמו ויכבשום לעבדים (ירמיה לד) : אבני קלע. האבנים הנזרקים בכף הקלע והוא הכלי המיוחד לזה : המו. מל' המייה : כמזרק. הוא הספל : כזויות. כקרנות : {טז} נזר. כתר כמו את הנזר (מ''ב י''א) : מתנוססות. מל' נס והוא ענין הרמה כמו נס להתנוסס (תהלים ס') : {יז} מה טובו. מלת מה יורה על הפלגת הרבוי כמו מה רב טובך (שם נ''א) : ותירוש. ענבי היין : ינובב. ענין דבור כמו ינוב חכמה (משלי י') :

זכריה פרק-י

{א} שאלו. עניז בקשה : מלקוש. כן יקרא המטר המאוחר כמו יורה ומלקוש (דברים י''א) : חזיזים. עננים כמו ודרך לחזיז קולות (איוב כח) : ומטר גשם. כפל המלה בשמות נרדפים וכן אדמת עפר (דניאל יב) : {ב} התרפים. עשוים הם בצורת אדם ומדברים עתידות ע''י כשוף : און. שקר ודבר תוהו : והקוסמים. הם חוזי הכוכבים : חזו. ראו : יענו. ענין הכנעה כמו מאנת לענות (שמות י') : רועה. ר''ל מנהיג : {ג} העתודים. הם הזכרים מן העזים ויאמר על השרים בדרך שאלה : אפקוד. ענין זכרון : הודו. ענין הדר ואמצות הלב : {ד} פנה. זויות : יתד. מסמר : נוגש. דוחק ולוחץ : {ה} בוסים. ענין רמיסה ברגל כמו יבוס צרינו (תהלים ס') : והובישו. מלשון בושה : {ו} והושבותים. היא מלה מורכבת מן הושבה ומן יושבת ההשקט מל' בשובה ונחת (ישעיה ל) : זנחתים. ענין עזיבה כמו למה זנחתנו (תהלים מ''ג) : ואענם. מל' עניה ותשובה : {ח} אשרקה. ענין צפצוף קול הבאה בקבוץ השפתים וכן ושרק לו מקצה הארץ (ישעיה ה').
