נחמיה פרק-א

{א} הבירה. ר''ל מקום ארמון המלך : {ב} הפליטה. ענין שארית וכן שריד ופליט (יהושע ח') : {ג} מפורצת. מל' פרצה ושבר : נצתו. מל' הצתה ושרפה כמו ויצת אש (איכה ד') : {ד} צם. מתענה : {ה} אנא. הוא ענין ל' בקשה : {ו} תהי נא. תהא עתה או היא ענין ל' בקשה : קשבת. ענין שמיעה : {ז} חבול. ענין השחתה כי השחית כל בשר (בראשית ו') ת''א חבילו : {ח} תמעלו. ענין חטא ופשע : אפיץ. אפזר :

נחמיה פרק-ב

{א} לארתחשסתא. הוא דריוש הפרסי ונקרא כן ע''ש המלכות כמו פרעה למצרים והדומים : {ג} ירעו. מל' רעה : חרבה. מל' חורבן : אוכלו. נשרפו : {ו} והשגל. המלכה המיוחדת לו למשכב המשגל : {ח} הפרדס. הוא גן מעץ פרי כמו גנות ופרדסים (קהלת ב') : לקרות. מלשון תקרה : הבירה. כן יקרא בה''מ כמ''ש לא לאדם הבירה (דה''א כ''ט) : {יב} רוכב בה. רוכב עליה וכן ינתן בו (שמות כ''א) ור''ל עליו : {יג} אכלו. נשרפו : {יד} ברכת. ענין מקוה המים וכן תעלת הברכה (ישעיה ז') : {טו} בנחל. בעמק ואמר עולה כי לפני הנחל היה הר : {טז} והסגנים. הוא מין שררה : ולחורים. כן יקראו השרים הגדולים והוא מל' חיור ולובן הפוך מן השפלים הנקראים חשוכים כמ''ש בל יתיצב לפני חשוכים (משלי כ' ב) : {יז} נצתו. מל' הצתה ושריפה :

נחמיה פרק-ג

{א} שער הצאן. כן שמו על דבר ידוע אצלם וכן כולם : {ה} ואדיריהם. מל' אדיר וחזק ור''ל העשירים : צורם. כמו צוארם באל''ף : {ז} פחת. ענין שררה ומושל : {ח} צורפים. צורפי זהב וכסף : ויעזבו. ענינו מלוי העפר לחזוק ובדרז''ל תקרה ומעזיבה (ב''מ קיו) : {ט} פלך. מחוז כי חבל ארגוב (דברים ג') ת''א בית פלך : {יג} השפות. כמו האשפות : {טו} ויטללנו. הוא ענין צל כי בצל קורתי (בראשית י''ט) ת''א בטלל ור''ל עשה חקרה לצל ומחסה : ברכת השלח. מקוה המים המשקה עצי הגן ונקראים שלח ע''ש התפשטות הענפים וכן שלחיך פרדס רמונים (ש''ה ד') : {יט} הנשק. כלי זיין כמו ועיר מבצר והנשק (מ''ב י') : המקצוע. הזוית כמו במקצוע החצר (יחזקאל מ''ו) : {כ} החרה. ענין חמום כמו נחר מפוח (ירמיה ו') : {כא} תכלית. קצה וסוף : {כד} הפנה. כן נקרא המגדל על כי דרכו לעמוד בפנה וכן ועל הפנות הגבוהות (צפניה א') : {כה} המטרה. הוא משמר בית האסורים כי ויאסוף אותם אל משמר (בראשית מ''ב) ת''א לבית מטרה : {כו} והנתינים. הם הגבעונים : בעופל. שם מגדל כמו ויבא אל העופל (מ''ב ה') : {לא} והרוכלים. הם סוחרי הבושם : {לד} האמללים. הנכרתים וחלושים כמו כי אומלל אני (תהלים ו') : היעזבו. מלשון מעזיבה והוא מלוי העפר לחזוק : היחיו. ענין רפואה כמו עד חיותם (יהושע ה') ובא בדרך השאלה וכמ''ש כי עלתה ארוכה לחומת וכו' (לקמן ד') : מערמות. ענין גל ותל וכן ויתנו ערמות ערמות (דה''ב ל''א) : {לז} תכס. כמו תכסה : אל תמחה. אל תמחוק :

נחמיה פרק-ד

{א} ארוכה. ענין רפואה כמו ארוכת בת עמי (ירמיה ח') ור''ל נתחזק ובא בדרך השאלה : {ב} ויקשרו. ענין אגודה : תועה. המבולבל בדעתו לבלי דעת מה יעשה קרוי תועה וכן תעיתי כשה אובד (תהלים קי''ט) : {ד} כשל. ענין חלשות ונפילת הכח כמו לכל כושל (דה''ב כ''ח) : הסבל. ענין טעינת משא כמו נושא סבל (מ''א ה') : {ה} והשבתנו. ענין בטול ומניעה כמו שבת נוגש (ישעיה י''ד) : {ז} בצחיחים. בעליונות הסלעים מקום היובש שבו והוא מלשון צחה צמא (שם ה') וכן על צחיח סלע (יחזקאל כ''ד) : רמחיהם. חניתות : {י} והרמחים. הוי''ו יתירה : והשריונים. הוא כעין מלבוש עשוי מברזל ללבוש במלחמה להציל ממכת חרב וחנית : {יא} עומשים. כמו עומסים בסמ''ך והוא ענין טעינת משא כמו ויעמוס איש על חמורו (בראשית מ''ד) : השלח. החרב כמו ויעש שלח לרוב (דה''ב ל''ב) : {יב} אסורים. קשורים : {יג} נפרדים. פזורים : {טו} השחר. הוא האור הנוצץ בפאת המזרח טרם יגלה השמש : {יז} שלחו. ענין הפשט כי ופשט את בגדיו (ויקרא ו) ת''א ושלח :

נחמיה פרק-ה

{ג} עורבים. ענין משכון כמו מה הערבון (בראשית לח) : {ד} למדת. כמו למנדת והוא מין מס כמו מנדה בלו והלך (עזרא ד') ונפלה הנו''ן : {ה} כובשים. ענין התפישה בחזקה כמו ויכבשום לעבדים (ירמיה לד) : לאל. ענין כח ואומץ כמו יש לאל ידי (בראשית ל''א) : {ז} וימלך. ענין עצה כמו מלכי ישפר עליך (דניאל ד) : משא. חוב והלואה כמו משאת מאומה (דברים כ''ד) : קהלה. אסיפה והקהל : {ח} כדי. מלשון די הפסוק : ויחרישו. ושתקו : {יא} ומאת הכסף. שקלי הכסף : התירוש. יין : היצהר. שמן : {יג} חצני. כן יקרא כנף הבגד וכן והביאו בניך בחוצן (ישעיה מ''ט) : נערתי. עניינו תנועת דבר מה בחזקה להפיל ממנו מה שעליו כמו התנערי מעפר (שם נ''ב) : {יד} פחם. פחה שלהם : {טו} אחר. הוא כענין מלבד ר''ל לבד לקיחת מ' שקלים לקחו עוד בלחם ויין : שלטו. משלו : {יח} נעשה. ענין תקון כמו וימהר לעשות אותו (בראשית י''ח) : ברורות. מל' ברירה ובחירה :

נחמיה פרק-ו

{א} פרץ. שבירה : {ב} ונועדה. מל' וועד ואסיפה כמו ויועדו כל המלכים (יהושע י''א) : {ג} תשבת. מלשון השבתה ובטול : ארפה. מלשון רפיון : {ו} וגשמו. הוא גשם הערבי וכן יתר יתרו : הוה. הוא מלשון הויה : {ז} נביאים. ענינו בעלי הדבור כמו ניב שפתים (ישעיה נ''ז) : {ח} בודם. ענין המצאת הכזב וכן בדיך מתים יחרישו (איוב י''א) : {ט} כולם. ענינו כמו כולכם ודוגמתו ואולם כולם תשובו (שם י''ז) : {י} עצור. מעוכב וכן נעצרה נא אותך (שופטים י''ג) : {יב} וסנבלט. הוי''ו היא במקום או כמו ומקלל אביו ואמו (שמות כ''א) : {טז} ויפלו. ענינו שפלות וגרעון כמו נופלים מערלים (יחזקאל ל''ב) :

נחמיה פרק-ז

{א} ויפקדו. מלשון פקידות וגזברות : {ג} יגיפו. יסגירו כי וסגרת הדלת (מ''ב ד') ת''י ותגיפון : {ה} להתייחש. ענין יחוס הוא הודעת ספר הדורות מן המשפחות ומי אביהם : {סד} ויגאלו. ענין טנוף ומאוס וכן לחם מגואל (מלאכי א') : {סו} רבוא. הוא עשרת אלפים : {סח} פרדיהם. הם הבאים מן הסוס והחמור : {ע} דרכמונים. שם מטבע ידוע אצלם : {עא} מנים. שם משקל ליטרא : {עב} ויגע. כמו והגיע :

נחמיה פרק-ח

{א} הסופר. החכם ובדח''ל סופר מברך ובור יוצא (חולין קי''ח) : {ה} עמדו. ענין שתיקה כמו עמדו ולא ענו (איוב ל''ב) : {ו} במועל. מלשון העלאה והרמה : ויקדו. כפפו הקדקוד : {ח} ושום. מלשון שימה : {י} מנות. ענין חלק ומתן כמו והיה לך למנה (שמות כ''ט) : {יא} מחשים. משתיקים כמו עת לחשות (קהלת ג') : הסו. שתקו כמו ויהס כלב (במדבר י''ג) : {טו} ויעבירו קול. הוא ענין הכרזה כמו ויעבירו קול במחנה (שמות ל''ו) : עבות. מלשון עב ור''ל מרובה בעלים וקלועים זע''ז וכן שתי עבותות הזהב (שם כ''ח) :

נחמיה פרק-ט

{ט} עני. מלשון עינוי : {יא} בקעת. חלקת לשנים : במצולות. עומק המים כמו ותשלך במצולות ים (מיכה ז') : עזים. מל' עוז וחזק : {יב} הנחיתם. הנהגתם כמו ולא נחם (שמות י''ג) : {טז} ערפם. הוא מול הפנים : {יז} וימאנו. מלשון מיאון וסירוב : {יח} מסכה. מדבר הניצק וניתך : נאצות. ענין כעס ובזיון : {כ} ומנך. מלשון מן : מנעת. חדלת : {כא} כלכלתם. ענין ספוק הצורך כמו ויכלכל יוסף (בראשית מ''ז) : בלו. מלשון בליה ורקבון : בצקו. ענין עיסה המחומץ : {כב} לפאה. לקצה כמו פאת שדך (ויקרא י''ט) : {כד} ותכנע. מלשון הכנעה : {כה} בצורות. מלשון מבצר : שמנה. מלשון שמן : ויתעדנו. ענין עונג ותפנוק כמו מעדני מלך (בראשית מ''ט) : {כו} גום. גופם כמו גוית שאול (שמואל א' ל''א) : {כז} ויצרו. מלשון צרה : {כח} וירדו. ומשלו כמו רדה בקרב אויביך (תהלים ק''י) : {כט} ותעד. ענין התראה : {לב} התלאה. ענין יגיעה וטורח כמו את כל התלאה (שמות י''ח) : {לה} ממעלליהם. ממעשיהם כמו עולל לי (איכה א') : {לז} גויותינו. ענין גוף :

נחמיה פרק-י

{א} אמנה. ענין קיום כמו יאמן נא דברך (מ''א ה') : החתום. ענינו גמר וסוף ובדרז''ל כל חותמי ברכות (ברכות ס''ג) : {ל} באלה. הוא קללת השבועה כמו בשבועת האלה (במדבר ה') : {לב} ונטש. ונעזוב כמו תשמטנה ונטשתה (שמות כ''ג) : ומשא. חוב ההלואה : {לד} המערכת. ענין סדור : {לה} מזומנים. ענין הכנה וקביעות וכן בעתים מזומנות (לקמן י''ג) : לבער. מלשון הבערה והסקה : {לח} עריסותינו. ענין עיסה כמו ראשית עריסותיכם (במדבר ט''ו) : תירוש. יין : ויצהר. שמן :

נחמיה פרק-יא

{א} הידות. החלקים כמו וארבע הידות (בראשית מ''ז) : {יז} יהודה. מלשון הודות והלל : {כג} ואמנה. ענין קיום :

נחמיה פרק-יב

{ח} הידות. מלשון הודות והלול : {כו} הפחה. המושל : הסופר. החכם : {כז} ובחנוכת. התחלת דבר מה לעמוד בה משם והלאה קרוי חנוך בל' מקרא : מצלתים וכו'. מיני כלי נגון : {כח} הככר. המישור : {לז} ועל, מעל. ענינם כמו לפני ואצל : {מד} הנשכות. כמו הלשכות : מנאות. מתנות וחלק כמו ושלחו מנות (לעיל ח') :

נחמיה פרק-יג

{ב} לא קדמו. לא באו לפניהם כמו במה אקדם ה' (מיכה ו') : {ה} ויעש. ענין תקון כמו וימהר לעשות אותו (בראשית י''ח) : {ו} נשאלתי. מל' שאלה ובקשה ועל כי שאלתו היה על עצמו אמר בלשון נפעל וכן נשאל נשאל דוד (שמואל א' כ') : {ז} נשכה. כמו לשכה : {יא} ואריבה. מלשון ריב : {יד} תמח. כמו תמחה והוא ענין מחיקה : {טו} גתות. הכלי שדורכים בו הענבים להוציא היין קרוי גת וכן כדורך בגת (ישעיה ס''ג) : הערמות. כרי לתבואה כמו ויתנו ערמות (ד''ה ב' ל''א) : ועומסים. ענין הטענת משא כמו ויעמוס איש על חמורו (בראשית מ''ד) : ציד. מלשון צידה ומזון : {טז} דג. ר''ל קבוצת דגים : {יז} חורי יהודה. שרי יהודה : {יט} צללו. מלשון צל : {כ} הרוכלים. מוכרי הבושם : {כא} לנים. מל' לינה : תשנו. מל' שנית : {כב} וחוסה. ענין חמלה כמו אתה חסת על הקיקיון (יונה ד') : {כה} ואמרטם. ענין תלישת השער כמו ואיש כי ימרט ראשו (ויקרא י''ג) : {כז} למעול. ענין חטא ופשע : {כט} גאלי. ענין טנוף ומאוס כמו ויגאלו מן הכהונה (לעיל ז') : {ל} משמרות. כל דבר המוטל על אנשים הרבה לעשות כל אחד בזמנו קרוי משמר כי ישמור לבוא בזמן הקבוע לו ולא יעבור : {לא} מזומנות. ענין הכנה וקביעות כמו לעתים מזומנים (לעיל י') :
