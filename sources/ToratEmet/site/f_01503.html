הושע פרק-א

{ב} זנונים. מל' זונה : זנה תזנה. ענין הסרה ועש''ז תקרא המנאפת זונה כי סרתה מדרכי הצניעות : {ד} ופקדתי. ענין זכרון כמו וה' פקד (ראשית כ''א) : והשבתי. ענין בטול כמו שבת נוגש (ישעיה י''ד) : {ה} קשת. ר''ל חוזק וגבורה : {ו} נשא אשא. ענין מחילת העון כמו ונשאתי לכל המקום (בראשית י''ח) : {ז} ובפרשים. הם הרגילים לרכוב על הסוסים : {ח} ותגמול. כן יקרא זמן השלמת היניקה כמו ביום הגמל את יצחק (שם כ''א) :

הושע פרק-ב

{א} ימד. מלשון מדידה : להם. עליהם : {ב} יזרעאל. היא מלה מורכבת מן זריעה ומן אל : {ד} אישה. בעלה : ונאפופיה. מלשון ניאוף : {ה} אפשיטנה. ענין הסרת המלבוש כמו ופשט את בגדיו (ויקרא ו') : והצגתיה. ענין העמידה כמו תציג אותו לבד (שופטים ז') : ושתיה. ענין שימה כמו למען שיתי (שמות י') : ציה. שממה : {ז} הובישה. מלשון בושה : הורתם. מלשון הריון : צמרי ופשתי. מלשון צמר ופשתים : ושקויי. מל' משקה : {ח} שך. ענין גדר כמו הסר משוכתו (ישעיה ה') : בסירים. קוצים כמו כקול הסירים (קהלת ז') : וגדרתי. מלשון גדר : ונתיבותיה. מלשון נתיב ושביל : {ט} תשיג. ענין הקרוב : אישי. בעלי : {י} והתירוש. כן יקרא ענבי היין : והיצהר. הוא השמן כמו תירושך ויצהרך (דברים י''א) : {יא} במועדו. בזמנו : והצלתי. ענין הפרשה כמו הלא אצלת לי ברכה (בראשית כ''ז) : ערותה. גלוי הבשר : {יב} נבלותה. ענינו דבר כעור ומגונה וכן והשלכתי עליך שקוצים ונבלתיך (נחום ג') : {יג} והשבתי. ענין בטול : משושה. שמחתה : {יד} והשמותי. מלשון שממון : אתנה. מלשון אתנן וכן יקרא מתן הזונה כמו אתנן זונה (דברים כ''ג) : {טו} ופקדתי. מלשון זכרון : ותעד. מלשון עדי קשוט כמו ועדית עדי (יחזקאל כ''ג) : נזמה. שם עדי כמו נזם זהב (משלי כ''ה) : וחליתה. ענין תכשיט מה כמו וחלי כתם (, שם) : {טז} מפתיה. מלשון פתוי והסתה : {יז} עכור. ענין השחתה כמו ועמק עכור (ישעיה ס''ה) : וענתה. ענינו הרמת קול שירה כמו ותען להם מרים (שמות ט''ו) : {כ} ורמש. הם בריות קטנות קצרי הרגל וכן רמש וצפור כנף (תהלים קמ''ח) : {כא} וארשתיך. מל' ארוס' : {כג} אענה. מלשון עניה והשבת שאלה :

הושע פרק-ג

{א} רע. חבר ועל הבעל יאמר ודוגמתו והיא חברתך (מלאכי ב') : פונים. מלשון הפנה וסבוב : אשישי. כוסות וגביעים כמו סמכוני באשישות (ש''ה ב') : ענבים. ר''ל יין ענבים : {ב} ואכרה. ענין קנין כמו וגם מים תכרו (דברים ב') : כסף. הוא מלה משותפת על הכסף ממש ועל התאוה כמו למעשה ידיך תכסוף (איוב י''ד) : וחומר. הוא הכור בת עשרה איפות והאיפה ג' סאים : ולתך. הוא חצי הכור חמשה איפות : {ג} תשבי. ענין עכבה והמתנה כמו אשב עד שובך (שופטים ו') : {ד} מצבה. הוא מזבח של אבן אחת ועושים אותה להקריב בה עכו''ם : אפוד. זה יאמר על האפוד והחשן עם האורים ותומים וכן אפוד ירד בידו (שמואל א' כ''ג) : {ה} אל ה'. את ה' וכן ואל טובו כמו וירדפו אל מדין (שופטי' ז') :

הושע פרק-ד

{ב} אלה. ענין שבועה : וכחש. שקר : פרצו. ענין שבר הגדר : {ג} תאבל. ענין השחתה כמו אבלה נבלה הארץ (ישעיה כ''ד) : ואומלל. ענין כריתה כמו כי אמלל אני (תהלים ו') : ואספו. ענין כליון ואבדון כמו אסף אסיפם (ירמיה ח') : {ד} אך. ת''י ברם ר''ל באמת וכן אך עצמי ובשרי אתה (בראשית כ''ט) : ירב. מלשון מריבה : {ה} ודמיתי. ענין כריתה כמו ער מואב נדמה (ישעיה ט''ו) : אמך. הכנסייא : {ז} כרובם. ענין גדולה כמו כי רבה היא (אסתר א') : בקלון. מלשון קלות ובזיון : אמיר. מלשון תמורה וחלוף : {ח} ישאו נפשו. זה הלשון יאמר על המצפה לדבר מה וכן מנשאים את נפשם (ירמיה כ''ב) : {ט} ופקדתי. ענין זכרון : ומעלליו. מעשיו : {י} יפרצו. ענין התחזקות הרבוי וכן פרצו לרוב (ד''ה א' ד') : {יא} ותירוש. הוא היין החדש : {יב} בעצו. מלשון עץ : ומקלו. מטה עץ : התעה. מלשון תועה : {יג} אלון ולבנה ואלה. שמות אילנות : {יד} יפרדו. ענין הפרשה כמו הפרד נא מעלי (בראשית י''ג) : הקדשות. הזונות כמו לא תהיה קדשה (דברים כ''ג) : ילבט. ענין כשלון ויגיעה כמו ואויל שפתי' ילבט (משלי י) : {טו} יאשם. מלשון אשמה : {טז} סוררה. ענין מרידה : ירעם. מל' מרעה : במרחב. במקום רחב : {יז} עצבים. כן יקראו העכו''ם : {יח} סר. מלשון הסרה : סבאם. כן יקראו הדברים המשכרים כמו סבאך מהול במים (ישעיה א') : הבו. ענין נתינה כמו הבו שכרי (זכריה יא) : קלון. ענין בזיון : מגניה. מלכיה : {יט} צרר. ענין קשירה כמו מי צרר מים (משלי ל) : בכנפיה. מל' כנף והמשילה לעוף שהרוח נושאה :

הושע פרק-ה

{ב} ושחטה. מל' שחיטה : שטים. ענין הסרה מדרך הטוב כמו ואת כי שטית (במדבר ה) : העמיקו. מל' עומק : מוסר. מלשון יסורין : {ג} נכחד. ענין מניעה כמו ולא כחדו מאבותם (איוב טו) : {ד} לא יתנו. לא יעזבו וכן ולא נתן סיחון (במדבר כא) : מעלליהם. מעשיהם : {ה} וענה. ענין הכנעה והשפלה כמו מאנת לענות מפני (שמות י) : גאון. מלשון גאוה ורוממות : {ו} חלץ. ענין השמטה והסרה כמו צדיק מצרה נחלץ (משלי יא) : {ז} חלקיהם. כן יקרא אחוזת השדה כמו ואכלה את החלק (עמוס ז) : {ח} חצוצרה. שם כלי נגון : הריעו. מל' תרועה והרמת קול : {ט} לשמה. מלשון שממון : תוכחה. מל' ויכוח : {י} כמסיגי גבול. אמר בל' שאלה כמי שמחזיר הסימן לתוך שדה חבירו להרחיב גבולו כמ''ש לא תסיג גבול רעך (דברים יט) : עברתי. מל' עברה וזעם : {יא} עשוק. ענין שדידה וגזל : רצוץ. ענין שבירה וכתיתה כמו עשוק ורצוץ (שם כח) : הואיל. ענין רצון כמו ויואל משה (שמות ב) : צו. מל' צואה : {יב} כעש. היא הרמה האוכלת הבגדים כמו כבגד יאכלם עש (ישעיה נא) : וכרקב. תולעת הגדלה בעץ נרקב : {יג} מזורו. כן נקרא המכה ע''ש שזורים עליה אבק מסממני' המרפאין וכן ישימו מזור תחתיך (עובדיה א) : ירב. הוא אשור ויקרא בשני השמות וכן מנחה למלך ירב (לקמן י) : יגהה. ענין רפואה כמו ייטב גהה (משלי יז) : {יד} כשחל. וככפיר. הם שמות האריה : {טו} ישחרונני. ענין דרישה כמו שוחר טוב (שם יא) :

הושע פרק-ו

{א} לכו. הוא ענין לשון זרוז : יך. מלשון הכאה : ויחבשנו. ענין קשירת המטלית על המכה והשבירה כמו ולנשברת לא חבשתם (יחזקאל לד) : {ב} יחיינו. ענין רפואה כמו עד חיותם (יהושע ה) : מיומים. ר''ל מהעתים : {ג} נרדפה. אמר בלשון שאלה על מרבית ההשתדלות : כשחר. היא אור הבוקר : נכון. ברור ומובטח : מוצאו. מל' יציאה : כמלקוש. הוא המטר המאוחר היורד על הקשין ועל המלילות כמו כעב מלקוש (משלי טז) : יורה. ענין שביעה כת''י והוא הפוך כמו ירוה שהוא ענין שביעה כמו כגן רוה (ישעיה נח) : {ה} חצבתי. ענין כריתת אבנים כמו חוצב בהר (מ''א ו) ואמר בל' שאלה על גזרת הדברים וחתוך האמרים : אור. דרך המקרא לכנות הטובה בשם אור וכמו שמכנה הרעה בשם חושך : {ח} עקובה. ענין מארב ואורב כמו ואת עקבו מים (יהושע ח) : {ט} וכחכי. ענין תקוה כמו וחכיתי לה' (ישעיה ח) : גדודים. מרבית אנשים יחד קרוי גדוד : חבר. מל' חבורה וסיעה : שכמה. ר''ל צד אחד ודעה אחת כמו לעבדו שכם אחד (צפניה ג) : זמה. מעשה תעוב ומגונה : {י} שערוריה. ענין דבר מגונה ומאוס כמו שמה ושערוריה (ירמיה ח) : {יא} שת. ענין שימה : קציר. ענף וסעיף כמו ועשה קציר (איוב יד) : שבות. מל' שבי הגולה :

הושע פרק-ז

{א} פשט. ענין הסרת המלבוש כמו ופשט את בגדיו (ויקרא ו) : {ג} ובכחשיהם. בשקריהם : {ד} בוערה. מל' הבערה והיסק : ישבות. ענין מניעה ובטול : מעיר. מלשון התעוררות : מלוש. מלשון לישה והוא עשיית העיס' כמו לושי ועשי עוגות (בראשית יח) : בצק. עיסה כמו את בצקו (שמות יב) : חומצתו. מל' חמץ : {ה} החלו. מל' חלי : חמת. הוא נאד וכן וחמת מים (בראשית כא) : לוצצים. מל' לץ : {ו} קרבו. ר''ל הכינו : בארבם. מל' אורב : אופיהם. מל' אפיה : {ז} יחמו. מל' חום : ואכלו. ר''ל הרגו : {ח} יתבולל. מל' בלבול וערבוב כמו שם בלל ה' (בראשית יא) : עוגה. חררה וכן ועשי עוגות (שם יח) : {ט} שיבה. הוא עוד יותר מזקנה : זרקה. ר''ל נזרקה : {י} וענה. ענין הכנעה כמו מאנת לענות מפני (שמות י') : גאון. מלשון גאוה ורוממות : {יא} פותה. מל' פתוי ולכן יקרא הסכל פתי כי הוא נפתה בקל : {יב} איסירם. מל' יסורין : {יג} נדדו. ענין הנעה : שוד. ענין עשק וגזל : אפדם. מלשון פדיון : {יד} יילילו. מל' יללה : על דגן. בעבור דגן : יתגוררו. מלשון גרות : יסורו. מל' סרירה ומרידה : {טו} יסרתי. ענין קשירה כמו כי אסר נאסרך (שופטים טו) והוא ענין חוזק כי קשירת הדבר הוא חזקו : {טז} על. מלשון עליון וכן הוקם על (ש''ב כ''ג) : כקשת רמיה. ר''ל איש רמיה כי דרך צייד הרמאי להפנות קשתו לצד אחר לבל תברח החיה ופתאום מהפכה מול החיה ויורה בה : מזעם. ענין קצף ועברה :

הושע פרק-ח
