עובדיה פרק-א

{א} וציר. ענין שליח כמו וציר אמונים (משלי יג) : {ב} בזוי. מבוזה : {ג} השיאך. ענין הסתה כמו הנחש השיאני (בראשית ג) : שוכני. היו''ד יתירה : בחגוי. ענין בקוע וחריץ כמו יונתי בחגוי הסלע (ש''ה ב) : {ד} קנך. מל' קן והוא מדור העופות : {ה} שודדי. ענין גזל ועושק : נדמיתה. ענין כריתה ואבדון כמו ער מואב נדמה (ישעיה טו) : דים. די הספוק כמו והמלאכה היתה דים (שמות לו) : בוצרים. כן יקראו תולשי ענבי כרם כמו כי תבצור כרמך (דברי' כד) : עוללות. הם הענבים הקטנים הגרועים כמו לא תעולל (, שם) : {ו} נחפשו. מלשון חפוש וחקירה כמו משם אחפש (עמוס ט) : נבעו. ענין דרישה וחפוש ודומה לו אם תבעיון בעיו (ישעיה כא) שהוא ענין דרישת שאלה וכמו שמלת בקשה הונח לדריש' שאלה ולחפוש כן מלת בעה הונח לשניהם : מצפוניו. מל' צפון ונסתר : {ז} שלחוך. ענין לויה כמו ואברה' הולך עמם לשלחם (בראשי' יח) : השיאך. ענין הסתה : יכלו. מל' יכולת ונצחון : לחמך. ר''ל אוכלי לחמך : מזור. ענין מכה ותקרא כן ע''ש שזורים עליה אבק מסממנים המרפאים וכן ולא יגהה מכם מזור (הושע ה) : תחתיך. במקומך : {ט} וחתו. ענין שבירה כמו והחתתי את עילם (ירמיה מט) : תימן. הוא אדום היושב בהר שעיר בדרומה של א''י : מקטל. ענין הרג כמו הן יקטלני (איוב יג) : {י} מחמס. ענין עושק : {יא} מנגד. ממרחק : שבות. מל' שבי : זרים. מלשון זר ונכרי : חילו. ענין עושר כמו ועזבו לאחרים חילם (תהלים מט) : ידו. ענין השלכה והפלה כמו ידו אליה (ירמיה נ) : {יב} נכרו. מל' נכר וזר : {יג} אידם. ענין צער ומקרה רע כמו הלא איד לעול (איוב לא) : בחילו. בעשרו : {יד} תעמוד. ענינו התחזקות על הדבר וההשתדלות עליו ודומה לו אך יונתן וגו' עמדו על זאת (עזרא י) : הפרק. ענין שביר' כמו ופרסיהן יפרק (זכריה יא). פליטיו. ענין שארית כמו פליט ושריד (איכה ב) : תסגר. ר''ל מסגר המאסר. שרידיו. ענין שארית : {טו} קרוב. ר''ל נכון ומזומן לפניו וכן כי קרוב יום אידם (דברים לב) : {טז} ולעו. ענין השחתה ובלבול וכן על כן דברי לעו (איוב ו) : {יז} מורשיהם. מל' ירושה : {יח} להבה. שלהבת : לקש. הוא תבן הדק : ודלקו. ענין שרפה והבערה : {יט} והשפלה. העמק : {כ} החל. כמו החיל ונפלה היו''ד :
