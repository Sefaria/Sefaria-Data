{א} ותקהל. נאספו : {ב} פנות. כמו בפנות, והוא מלשון פנה וזוית : רגלי. הולכי רגל, לא רוכבים : שולף חרב. רוצה לומר, בחרב שלופה : {ה} עלי. בעבורי : דמו. חשבו, כמו (ישעיהו י ז) והוא לא כן ידמה : ענו. הוא האונס, כמו (דברים כב כד) על דבר אשר ענה : {ו} זמה. מחשבה רעה, כמו (איוב לא יא) כי הוא זמה, וזהו מה שדימו להרגו : {ז} הבו. תנו : הלום. פה, כמו (שמות ג ה) אל תקרב הלם : {ח} נסור. מלשון הסרה : {י} לרבבה. היא עשרת אלפים : צדה. מזון : לבואם. הלמ''ד במקום בי''ת, וכן (שמואל א כ ל) לבן ישי : {יב} מה הרעה. מלת 'מה' מורה על גודל הדבר והפלגתו, כמו (תהלים לא כ) מה רב טובך : {יג} ונבערה. ענין פינוי וכליון מן העולם, כמו (דברים יג ו) ובערת הרע מקרבך : אבו. רצו : {טו} ויתפקדו. מנין : {טז} בחור. כל אחד היה נבחר : אטר וכו'. רוצה לומר, שמאלי, ודוגמתו באהוד (לעיל ג טו) : קלע. ענין השלכת אבנים בכף הקלע, והוא הכלי העשוי לכך : השערה. חוט השער : יחטיא. ענין חסרון, כמו (מל''א א' כ''א) והייתי אני ובני שלמה חטאים : {כו} וישבו. ענין עכבה : ויצומו. התענו : {כח} אחדל. אמנע : {כט} ארבים. יושבים במארב ונסתרים : {לא} הנתקו. נעתקו וסרו : ויחלו. מלשון התחלה : במסלות. דרך כבושה : {לב} נגפים. מוכים, כמו (תהלים צא יב) פן תגוף באבן רגלך : {לג} מגיח. ענין משיכה והוצאה, כמו (תהלים כב י) אתה גוחי מבטן : ממערה. ממקום גלויה שנוח לכבשה משם, והוא כמו (ויקרא כ יח) מקורה הערה : {לז} החישו. מהרו, כמו (תהלים נה ט) החישה מפלט לי : ויפשטו. ענין פזור, וכדרך הנלחמים : לפי. חדוד החרב תקרא פה : {לח} והמועד. הזמן, כמו (ירמיהו מו יז) העביר המועד, ורוצה לומר, הודעת זמן בואם : הרב. הרבה : משאת העשן. עמוד העשן, על שם שנשאת כלפי מעלה : {מ} החלה. התחילה : עמוד עשן. אמר לתוספת ביאור, כי כבר אמר 'והמשאת' : כליל. מלשון כל, כמו (ויקרא ו טו) כליל תקטר, ורוצה לומר, עשן שריפת כל העיר : {מב} הדביקתהו. השיגה אותו והיה דבוק אליו : {מג} כתרו. מלשון כתר : הדריכוהו. מלשון דריכה ורמיסה : {מה} ויעללהו. מלשון עוללות, והם הנלקטים אחר הבציר, כמו (לעיל ח ב) הלא טוב עללות אפרים : {מח} מתם. הוא שם לאנושיות, וכולל אנשים ונשים, כמו (בראשית לד ל) מתי מספר : שלחו. רוצה לומר, הבעירו, על שם שהאש עושה השליחות שהולך ושורף :
