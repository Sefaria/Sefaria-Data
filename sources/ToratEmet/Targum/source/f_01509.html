צפניה פרק-א

{ב} אסוף אסף. ענין כליון כמו ואספת את נפשך (שופטים יח) והיה ראוי לומר אסף אאסף ותחס' האל''ף : {ג} את הרשעים. עם הרשעים : {ד} שאר. שיור : הכמרים. כן יקראו כהני עבודת כוכבים וכן וכמריו עליו יגילו (הושע י) : הכהנים. הם מין אחר מתחלפים בלבושיהם : {ו} הנסוגים. ענין החזרה לאחור כמו ולא נסוג ממך (תהלים פ) : {ז} הס. ענין שתיקה כמו ויהס כלב (במדבר יג) : הקדיש. ענין הזמנה כמו קדשו קהל (יואל ב) : קרואיו. המזומנים לסעודה וכן בראש הקרואים (ש''א ט) : {ח} ופקדתי. ענין השגחה : נכרי. ענין זרות : {ט} הדולג. ענין קפיצה : המפתן. אסקופה כמו מתחת מפתן הבית (יחזקאל מז) : חמס. עושק : {י} המשנה. מל' שנים ור''ל בין שתי החומות שהיא משנה לעיר וכן והיא יושבת בירושלים במשנה (מ''ב כב) : {יא} הילילו. מל' יללה : המכתש. שם מקום בירושלים עמוקה ככלי המכתשת שדוכין בה בשמים : נדמה. ענין כריתה כמו ער מואב נדמה (ישעיה טו) : עם כנען. ר''ל עדת התגרים כמו כנען בידו מאזני מרמה (הושע יב) : נטילי. ענין משא כמו ונטל החול (משלי כו) : {יב} אחפש. ענין בקור ודרישה כמו תמנו חפש מחפש (תהלים סד) : הקופאים. הצפים ממעל כי ויצף הברזל (מ''ב ו) תרגומו וקפא פרזלא : שמריהם. הוא עכירת המשקה הנופל לשולי הכלי כמו ושקט הוא אל שמריו (ירמיה מח) : ירע. מלשון רעה : {יג} חילם. עשרם כמו ישא את חיל דמשק (ישעיה ח) : למשסה. ענין רמיסה כמו למשסה ואין אומר השב (שם מב) : {יד} צורח. ענין שאגת קול גדול כמו יריע אף יצריח (, שם) : {טו} עברה. ענין כעס : ומצוקה. עניינו כמו צרה וכן בארץ צרה וצוקה (שם ל) וכפל המלה בשמות נרדפים להפלגות הענין : שואה ומשואה. ענין חושך ושממון כמו אמש שואה ומשואה (איוב ל) : וערפל. הוא עב הענן : {טז} הפנות. הם המגדלים שדרכם לעמוד בפנות וזויות המבצר : {יז} והצרותי. מל' צרה : ולחומם. כן יקרא גם הבשר כמו כי לחמם לנפשם (הושע ט) : כגללים. כצואה כמו כגללו לנצח יאבד (איוב כ) : {יח} קנאתו. ענין כעס וחמה כמו הם קנאוני בלא אל (דברים לב) : תאכל. ענין השחתה : נבהלה. מל' בהלה :

{א} מוראה ונגאלה. ענין שניהם מלוכלך ומטונף כמו ושמתיך כרואי (נחום ג) וכמו נגאלו בדם (ישעיה נט) וכפל המלה בשמות נרדפות : היונה. מל' אונאה ועניינו קחת ההון וכן מפני חרון היונה (ירמיה כה) : {ג} גרמו.
